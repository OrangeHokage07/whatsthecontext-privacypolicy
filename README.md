# whatsthecontext-privacypolicy
What's the Context?  Privacy Policy


DATA WE STORE LOCALLY

- Text you copy or highlight on webpages
- Source URL, page title, and timestamp of each item
- Your AI provider name and model preference
- Your AI API key (stored only in your browser's local storage)
- Saved browser sessions and tab lists


All of the above is stored exclusively on your device using chrome.storage.local. It never leaves your device except as described below.


DATA WE TRANSMIT
When you explicitly click Summarize, Rephrase, Proofread, or Translate, the selected text is sent to the AI provider you configured (OpenAI, Google Gemini, or Anthropic) over HTTPS. 
No other data is transmitted. We do not operate any servers and do not receive any data ourselves.


DATA WE DO NOT COLLECT
- We do not collect analytics
- We do not collect crash reports
- We do not share data with third parties
- We do not transmit your API key to anyone except the provider's own API endpoint over HTTPS


DATA DELETION
You can delete all stored data at any time by clicking "Clear & Remove Key" in the extension settings panel, or by removing the extension from Chrome entirely.
