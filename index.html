What's the Context?

Privacy Policy

Last updated: February 2026

Overview

What's the Context? is a Chrome extension that helps you copy, highlight, and AI-process text from any webpage. We are committed to protecting your privacy. This policy explains what data is stored and how it is used.

Data Stored Locally on Your Device

The following data is stored exclusively in your browser using chrome.storage.local. It never leaves your device except as described in the section below.

  • Text you copy or highlight on webpages
  • Source URL, page title, and timestamp of each item
  • Your AI provider name and model preference
  • Your AI API key
  • Saved browser sessions and tab lists

Data We Transmit

When you explicitly click Summarize, Rephrase, Proofread, or Translate, the selected text is sent over HTTPS directly to the AI provider you configured — OpenAI, Google Gemini, or Anthropic. This request is made from your browser directly to the provider's API.

We do not operate any servers. We do not receive or store any data ourselves. Only the text you explicitly submit for AI processing is transmitted, and only to the provider whose API key you have entered.

Data We Do Not Collect

  • We do not collect analytics or usage statistics
  • We do not collect crash or error reports
  • We do not share any data with third parties
  • We do not transmit your API key to anyone except the provider's own API endpoint
  • We do not track browsing history

Permissions Used

  • storage — saves clipboard items, highlights, sessions, and settings locally
  • activeTab — reads the current page's URL and title to tag saved items
  • sidePanel — opens the extension's side panel
  • scripting — injects the text-selection bubble into webpages
  • tabs — reads open tabs for the Save Session and Group Tabs features
  • tabGroups — creates Chrome tab groups when you use Group Tabs
  • alarms — schedules cleanup of expired local items every hour
  • host permissions (<all_urls>) — required so the text-selection bubble works on every website you visit

Data Deletion

You can delete all locally stored data at any time by clicking "Clear & Remove Key" in the extension settings panel, or by removing the extension from Chrome entirely via chrome://extensions.
